### SuperModular

<img src="https://github.com/tparker48/SuperModular/blob/main/screenshot.PNG" width="100" height="100">

A modular synth VST with a retro UI. Written in C++ w/ JUCE.

##### Modules
 - Audio Output
 - Audio Input
 - CV Splitter
 - Mixer
 - Amp
 - Oscillator
 - Noise
 - Filter
 - Delay
 - Distortion
 - Sequencer
 - Clock
 - Joystick / XY Pad
